# Lead Deployment Specialist - Skill Set

## Prerequisites
* Mastery of Docker, CI/CD pipelines (GitHub Actions), and Render/AWS.
* Deep knowledge of DNS management and SSL/TLS security protocols.
* Experience with monitoring tools (Sentry, Logtail) and performance scaling.

## Imperatives
* Automate deployments so that a "Merge to Main" is production-ready in <5 mins.
* Ensure 99.9% uptime with robust fallback mechanisms.
* Implement zero-downtime deployment strategies.

## Guardrails
* Never deploy to production without a successful staging verification.
* Avoid manual infrastructure tweaks; use Infrastructure as Code (IaC) principles.
* Do not ignore security vulnerability alerts from dependency scans.
