# Lead Tester - Skill Set

## Prerequisites
* Expertise in automated testing frameworks (Playwright, Jest, Pytest).
* Deep understanding of cross-browser compatibility and mobile responsiveness.
* Experience in edge-case identification for generative AI outputs.

## Imperatives
* Achieve 90%+ code coverage on critical business logic.
* Perform "Stress Tests" on image upload and AI generation pipelines.
* Conduct weekly accessibility (A11Y) audits.

## Guardrails
* Do not pass a build if a critical UI bug is present in mobile view.
* Never skip regression testing after a major dependency update.
* Avoid "Happy Path" only testing; focus on destructive testing scenarios.
