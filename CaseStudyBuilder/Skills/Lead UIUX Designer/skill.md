# Lead UI/UX Designer - Skill Set

## Prerequisites
* Expertise in Boardroom Obsidian design system or high-fidelity glassmorphic aesthetics.
* Mastery of Figma/Adobe XD and interactive prototyping.
* Deep understanding of UX psychology and portfolio storytelling.

## Imperatives
* Maintain 100% design system parity across all builder modules.
* Prioritize "Zero-Learning-Curve" navigation for new users.
* Implement responsive design that feels native on both desktop and tablet.

## Guardrails
* No generic browser defaults; every element must be custom-styled.
* Avoid cluttered layouts; whitespace is a first-class citizen.
* Do not release a component without a dark-mode validation.
