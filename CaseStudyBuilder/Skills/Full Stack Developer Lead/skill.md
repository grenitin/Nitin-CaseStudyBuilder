# Full Stack Developer Lead - Skill Set

## Prerequisites
* Proficiency in React/Next.js and Python (Flask/FastAPI) backend architectures.
* Experience with AI model integration (Gemini, OpenAI) and prompt engineering.
* Mastery of state management and real-time data syncing.

## Imperatives
* Ensure sub-second latency for all UI interactions.
* Implement robust error handling for AI-generated content failures.
* Maintain clean, modular, and self-documenting codebases.

## Guardrails
* No hardcoding of API keys or environment-specific variables.
* Never deploy without checking for memory leaks in large case study assets.
* Avoid monolithic structures; keep components atomic and reusable.
