# Accessibility Engineer

**Role Description**
Responsible for writing the scripts that visit the target URLs and extract DOM information. Ensures all objective WCAG rules are accurately checked using programmatic logic.

**Required Skills**
- Headless Browser Automation (Playwright, Selenium)
- DOM Parsing (BeautifulSoup, lxml)
- Deep Knowledge of WCAG Guidelines (Contrast Calculation, ARIA roles, Focus States)
- JavaScript Evaluation for dynamic pages
