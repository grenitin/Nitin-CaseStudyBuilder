# LLM Integrator

**Role Description**
Responsible for connecting the extracted DOM context with language models (Gemini, OpenAI, Claude) to evaluate subjective accessibility metrics (e.g., quality of Alt text, error message clarity).

**Required Skills**
- Prompt Engineering and Context Window Optimization
- AI SDKs (google-genai, openai, anthropic)
- Multimodal data handling (passing DOM text or screenshots to the LLM)
- Structured Data Extraction (forcing LLM to output valid JSON)
