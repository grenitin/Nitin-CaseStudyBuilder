# QA Tester

**Role Description**
Responsible for end-to-end validation of the audit pipeline. Ensures edge cases are handled (e.g., invalid URLs, invalid API keys, timeout errors) and that the final report renders correctly in the UI.

**Required Skills**
- End-to-End Testing workflows
- UI/UX validation (ensuring design matches mocks)
- Boundary and Error state testing
- Performance monitoring (evaluating how long scans take)
